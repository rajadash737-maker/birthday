# 🎂 Happy Birthday Mini Website 🎉

A beautiful, interactive mini website built to celebrate a loved one in a personal way.
The project uses **HTML, CSS, and JavaScript** with animations, surprises, and heartfelt wishes.

Anyone can customize this and create a unique online birthday greeting.

---

## 🚀 Live Demo
👉 [Click here to view the site](https://nikitayadav19.github.io/HappyBirthdayGF)

---

## 🎁 Features
- 💌 Heartfelt birthday message with typewriter effect
- 🌟 Responsive design (works on phone and desktop)
- ✨ Easy to customize for anyone
- 🎨 Theme switcher (Romantic / Dreamy / Golden)
- 🎆 Confetti bursts + final fireworks surprise page
- 🖼️ Auto memory slideshow + lightbox photo preview
- 🎤 Voice-note button with speech fallback
- 💌 Typing love-letter modal

---

## 🛠️ Tech Stack
- **HTML5**
- **CSS3** (animations + gradients)
- **JavaScript**

---

## 🎉 How to Use
1. Clone the repo:
```bash
git clone https://github.com/nikitayadav19/HappyBirthdayGF.git
```
2. Open the project folder.
3. Launch `index.html` in your browser.
4. Edit text/images to personalize for your person.

### 🎵 Optional: Add Real MP3 Background Song
1. Put your audio file in project root as `birthday-song.mp3`.
2. Open the site and click anywhere once (browser audio policy).
3. Use `Play Birthday Song` button to play/pause.

If `birthday-song.mp3` is not present, the app automatically uses built-in melody + voice lyrics.

### 🎙️ Optional: Add Personal Voice Note
1. Put your voice message in project root as `voice-note.mp3`.
2. Click `Play My Voice Note` on the memories page.
